
# Home Assistant JBD BMS Bluetooth Integration

这是一个为 JBD (小象/中颖/蚂蚁) 保护板开发的 Home Assistant 原生集成。它通过蓝牙直接连接（支持 ESPHome 蓝牙代理），无需额外的串口转 WiFi 模块。

## 🌟 核心特性

- **🚀 原生蓝牙代理支持**：完美支持 ESPHome Bluetooth Proxy，解决远距离连接难题。
- **🛡️ 状态缓存技术**：深度优化了蓝牙漏包导致的“实体不可用”问题。即使单次读取失败，也会显示上次缓存值，界面不闪烁。
- **⏱️ 库仑计预估**：内置软件算法，根据实时电流和容量自动计算**预估充满/用完时间**。
- **🔋 动态电芯支持**：自动识别 4S - 32S 电池组，动态生成每一串电芯电压及**均衡状态实体**。
- **⚙️ 全功能控制**：支持充电/放电 MOS 开关，并可在 HA 中直接修改**均衡模式**（关闭/充电均衡/静态均衡）。

## 📊 支持实体

### 传感器 (Sensors)
- **核心数据**：总电压、电流、功率、剩余电量 (%)、剩余容量 (Ah)、循环次数。
- **分析数据**：单体平均电压、最大压差、最高/最低电压电芯编号。
- **时间预估**：预估剩余时间 (Duration)、时间预估状态。
- **诊断文本**：运行状态 (充电/放电/待机)、当前告警信息 (中文字符串)。

### 二元传感器 (Binary Sensors)
- **保护告警**：过压、欠压、过温、短路等 13 项保护状态监测。
- **均衡状态**：每一串电芯的均衡状态 (Balancing)。

### 控制实体 (Controls)
- **开关 (Switch)**：充电 MOS 开关、放电 MOS 开关。
- **配置 (Select)**：均衡模式切换。
- **按钮 (Button)**：BMS 系统重启。

## ⚙️ 安装步骤

### 1. 手动安装
1. 下载仓库代码。
2. 将 `custom_components/jbd_bms_ble` 文件夹复制到你的 Home Assistant 配置目录下的 `custom_components` 文件夹中。
3. 重启 Home Assistant。

### 2. 配置
1. 在 Home Assistant 中进入 **配置 > 设备与服务**。
2. 点击 **添加集成**，搜索 `JBD Smart BMS`。
3. 输入保护板的蓝牙 MAC 地址 (可以在官方 App 或 ESPHome 日志中查看)。
4. 建议在集成选项中设置 `读取后断开连接延迟` 为 5-10 秒，以保证官方 App 仍能连接。
